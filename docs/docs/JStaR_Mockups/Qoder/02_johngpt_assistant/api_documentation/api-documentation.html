Chat Completion

Generate intelligent responses using JohnGPT's business personas

POST /api/v2/chat/completions

Parameters

Parameter Type Required Description
messages array required Array of message objects
persona string optional Business persona to use (entrepreneur, marketer, coach, etc.)
temperature float optional Response creativity (0.0 - 2.0, default: 0.7)
max_tokens integer optional Maximum response length (default: 2048)

Example Request

{ "messages": [ { "role": "user", "content": "Help me create a marketing strategy for my new SaaS product" } ], "persona": "marketing_expert", "temperature": 0.7, "max_tokens": 1500 }

Response

{ "id": "chatcmpl-7QyqpwdfhqwajicIEznoc6Q47XAyW", "object": "chat.completion", "created": 1677858242, "model": "johngpt-business-v2", "persona": "marketing_expert", "choices": [ { "index": 0, "message": { "role": "assistant", "content": "Here's a comprehensive marketing strategy for your SaaS product..." }, "finish_reason": "stop" } ], "usage": { "prompt_tokens": 56, "completion_tokens": 424, "total_tokens": 480 } }
{ "error": { "message": "Invalid persona specified", "type": "invalid_request_error", "code": "invalid_persona" } }
