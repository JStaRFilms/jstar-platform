## Core Frameworks
- **The Number Hook**: `5 [Adjective] [Noun] That [Benefit]`
  - Example: "5 Hidden YouTube Features That Skyrocket Your Views"
  - Best for: Tutorials, listicles
  
- **The Curiosity Gap**: `[Question]? Here's Why...`
  - Example: "Why Do Viral Videos Always Start Like This? (Data-Backed)"
  - Best for: Analysis, thought leadership

2. Track CTR in your [[YouTube Analytics Dashboard]]
